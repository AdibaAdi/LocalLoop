# LocalLoop

LocalLoop is a full-stack-ready civic engagement web app scaffold built with **React + Vite + Tailwind CSS** on the frontend and **Firebase** integrations (Auth, Firestore, Storage, Cloud Messaging) in the client layer.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Copy env template:
   ```bash
   cp .env.example .env
   ```
3. Fill in your Firebase credentials in `.env`.
4. Start dev server:
   ```bash
   npm run dev
   ```
Testing

    Built a production bundle with npm run build and the build completed successfully.
    Ran static checks with npm run lint and lint succeeded.
